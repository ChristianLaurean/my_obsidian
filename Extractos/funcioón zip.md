>[!tip] Sirve para unir dos [[Listas]] interlazando los elementos por [[Tuples]]
>```python
>nombre = ['Ana','hugo','Valeria']
>edad = [65,29,42]
>conbinados = list(zip(nombre,edad)) # Se interlazan los datos en una tupla [('Ana',65)]
