# Función random

Nos sirve para tener números aleatorios
```python
from random import randint #integer aleatorio
from random import uniform # datos floats aleatorio
from random import choise # aleatorio con las listas
from random import shuffle # ordena de forma aleatoria las listas
aleatorio = randint(1,50)
print(aleatorio)
```
