# Formatear cadenas

>[!tip] Para que sirve?
>Es una forma de ahorrarte tiempo a la hora de concatenar datos

### Format

```python
print("Mi auto es {} y cuesta: {}".format(variable_autho,variable_costo))
```

### F string

```python
print(f"Mi auto es {variable} y cuesta:{variable}")
```
