# Crear carpetas

`mkdir nombre_carpeta`

# Crear archivos

`touch nombre_archivo`

# Eliminar carpeta
- Para carpetas que estén vacías
`rmdir nombre_carpeta`
- Para eliminar carpetas con contenido 
`rm -ri nombre_carpeta
# Eliminar archivos
`rm nombrearchivo`
