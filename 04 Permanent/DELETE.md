# delete
>[!tip] Borrar registros de una tabla


>[!warning] Antes de usar DELETE!!!!
>Primero debemos de hacer una consulta con el [[WHERE]] y [[SELECT]] y ver que nos esta devolviendo y de ahi eliminar
>```SQL
>SELECT * FROM user
>WHERE id = 2
>
>
>DELETE FROM  user WHERE id = 2
>```
